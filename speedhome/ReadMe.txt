This application is developed on Spring Boot.
MySQL database used.
All the libraries used in this application are open source. No specific license is required.
Table structure and input json required for testing also available at root directory.
Below mentioned properties need to be update at application.properties available at speedhome/src/main/resources
spring.datasource.url=jdbc:mysql://localhost:3306/speedhome
spring.datasource.username=
spring.datasource.password=
Attributes used here are taken from speedhome.com search module which used for property search
